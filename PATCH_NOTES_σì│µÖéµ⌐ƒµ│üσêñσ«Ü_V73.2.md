# PATCH NOTES - 即時機況判定 V73.2

## 修改目的
依最新指定口徑修正設備即時機況判定：

1. 最新架設場次未填後校正日：運作中。
2. 最新架設場次已有後校正日，且後續尚未有新增架設場次：維護中。
3. 「有後校正日但尚未逾 2 日」也判定為維護中，不再保留等待期。

## 調整檔案
- `server.js`
- `package.json`

## 主要影響
- LINE 輸入「即時機況」時，狀態會立即依新標準顯示。
- 後台「設備即時機況」同步結果同步套用。
- 原本 `LIVE_EQUIPMENT_WAIT_NEW_SESSION_DAYS=2` 不再作為「尚未逾 2 日仍運作中」的判定依據；保留環境變數不影響啟動。

## 判定摘要
| 條件 | 結果 |
|---|---|
| 無架設回報 | 維護中 |
| 有最新架設回報，無後校正日 | 運作中 |
| 有後校正日，且後續無新增架設場次 | 維護中 |
| 最新架設場次日不早於後校正日 | 運作中 |
